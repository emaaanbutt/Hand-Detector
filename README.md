
🖐 Hand Detection with MediaPipe & OpenCV
===========================================

📌 About
--------
Detects hands in real-time using MediaPipe and OpenCV. 
Shows 21 landmarks per hand with connections.

-------------------------------------------
📂 Requirements
-------------------------------------------
pip install opencv-python mediapipe

-------------------------------------------
▶️ Run
-------------------------------------------
python hands.py

Press q to exit.
